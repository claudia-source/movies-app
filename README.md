# 🎬 Movies App

Aplicación web fullstack para gestionar películas. Permite listar, agregar y administrar películas mediante una API conectada a una base de datos.

---

## 🚀 Tecnologías utilizadas

### 🔹 Frontend

* React
* Axios
* Bootstrap

### 🔹 Backend

* Node.js
* Express
* MySQL / MongoDB

---

## 📁 Estructura del proyecto

```
movies/
├── backend/
├── frontend/
└── README.md
```

---

## ⚙️ Instalación y ejecución

### 🔹 Clonar el repositorio

```bash
git clone https://github.com/TU_USUARIO/movies-app.git
cd movies
```

---

### 🔹 Backend

```bash
cd backend
npm install
npm run dev
```

Servidor en:

```
http://localhost:3001
```

---

### 🔹 Frontend

```bash
cd frontend
npm install
npm start
```

Aplicación en:

```
http://localhost:3000
```

---

## 🔗 Conexión entre frontend y backend

El frontend consume la API desde:

```
http://localhost:3001/api
```
